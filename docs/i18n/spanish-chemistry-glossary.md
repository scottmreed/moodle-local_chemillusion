# Glosario español de química y producto (ChemIllusion + Moodle)

Glosario de referencia para mantener una terminología **consistente** entre el
sitio web de ChemIllusion, el complemento de Moodle y la documentación. Usa
**español neutro de América Latina**. Los tokens técnicos (SMILES, InChI,
InChIKey, PubChem, RDKit, WASM, CID) no se traducen.

> Este archivo es la fuente de verdad compartida. Existe una copia equivalente
> en `chem-art-generator` (`docs/i18n/spanish-chemistry-glossary.md`).

## 8. Guía de tono

Usar:

- Español neutro de América Latina, claro y directo.
- "Profesor/profesora" o "docente" según el contexto.
- "Estudiante" en vez de "alumno" para mayor neutralidad.
- "Complemento de Moodle" para el plugin.
- "Código abierto" para open source.
- "Cuenta gratuita" para free account.

Evitar:

- Jerga literal de SaaS.
- Términos demasiado específicos de España.
- Orden de palabras propio de la traducción automática.
- "Créditos de IA" como título sin explicación.
- Copia legal/privacidad compleja copiada del inglés.
